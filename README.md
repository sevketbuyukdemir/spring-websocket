# Spring WebSocket

WebSocket integration implementation between Next.JS (SockJS Client) and Spring Boot (Server)

## Technologies
- Java 17
- Maven
- Spring Boot
- [Spring WebSocket](https://docs.spring.io/spring-boot/reference/messaging/websockets.html)
- [SockJS](https://github.com/sockjs)

## Running the Application

1. Clone repository
```
git clone https://github.com/sevketbuyukdemir/spring-websocket.git
```
2. Run Spring Boot websocket-server application (You can open it in IntellijIdea)
3. Run Next.js Client Application and go to the [browser](http://localhost:3000) to test.
- Open client-application folder in VS Code, then run the commands below.
```
npm install
```
```
npm run dev
```
